# Quick Pass

Scanne. Vérifie. Entre.

## Description

Quick Pass est une application de billetterie QR minimale mais solide. Crée des événements, génère des billets avec QR, imprime les tickets, scanne à l’entrée, et conserve un historique d’usage. Interface mobile-first avec une navigation bottom-bar en 3 onglets (Générer — Scanner — Historique) et un dashboard admin pour gérer événements, invités et rapports. Sécurité : vérification côté serveur, journalisation des scans, prévention des races conditions.
